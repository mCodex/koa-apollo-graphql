# Koa GraphQL Apollo Babel

My KOA stack starter including:

* KOA
* GraphQL
* Apollo
* Babel 7
* ESLint
* Nodemon

## Installation

Clone or download this repository and run:

```bash
yarn
```

And then:

```bash
yarn start
```

After that, open: http://localhost:4000/graphql

## Production Build

```bash
yarn build
```

And then your build will be available inside `dist` folder.
